# RClass2026 - R语言与数据可视化课程

## 课程简介

本项目包含2026年R语言与数据可视化课程的所有教学材料。

## 项目结构

- `lessons/` - 课程教案和练习
- `data/` - 课程使用的数据集
- `scripts/` - R脚本文件
- `outputs/` - 生成的图表和报告

## 使用说明

1. 克隆项目到本地
2. 打开RStudio项目文件
3. 根据课程进度学习相应章节
